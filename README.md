# NameGenerator

<p>A simple tool for generate Chinese or American or Japanese Names.<p>
<p>It requires Python Qt5 package, you should install it with the command:<p>
<p><b>pip install pyqt5</b></p>
<p>Then run it:</p>
<p><b>python nameGenerator.py</b></p>
